## 项目规范

1. 添加新元素或元素的新属性时，需要注意 codeGenerater 中也需要做出处理
2. 为元素添加新 setting config 时，如果是元素上的属性配置，则需要判断是否需要对特殊字符进行转义处理
3. 对于可以嵌套内容的元素，需要按照 HTML 规范，定义元素的 exclude
4. 画布中一些特殊的可以进行互动的元素，在画布处于编辑状态时，都必须阻止其默认交互行为。例如：聚焦在 form 元素内的输入框时，按下 enter 不能够发出请求。但在画布预览模式下允许互动
5. 增加元素的属性时，需要在组件定义处将属性也注入进去，不能够只有在生成的代码中使用
6. 添加新元素时，若新元素可以/不可以为已有画布元素的子元素，则需要在已有画布元素的 include/exclude 中添加新元素
